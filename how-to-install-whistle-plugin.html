# 如何安装 whistle 插件

该功能尚在 beta，需要的话请在设置界面中升级到 beta 版本提前使用

​​​​​

以 whistle.helloworld 为例，安装 Nodejs 后，直接使用 npm 全局安装该模块即可。


例如：


npm install -g whistle.helloworld
